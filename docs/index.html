なによりも素早く

ネットに接続しないことによって、高速な検索を実現。記号を最も速く検索できます。

なによりも楽に

アプリを開いて見たままに記号を書くだけ。記号の読み方を知っている必要はありません。

どんな記号でも

ISO規格から、数学・化学・物理分野などで使われる記号を独自のデータベースで用意。

小さなアプリサイズ

オフライン辞書でアプリが重くなることはありません。記号専用の辞書なので、小さいアプリサイズを実現。
